Sobre a Disciplina

Fábrica de Projetos Ágeis é uma disciplina focada no desenvolvimento de habilidades essenciais para a gestão de projetos de forma eficiente e adaptável, utilizando metodologias ágeis. Através dessa disciplina, os alunos aprendem a aplicar frameworks como Scrum, Kanban e Lean, ferramentas amplamente adotadas por empresas que buscam otimizar processos, reduzir custos e aumentar a satisfação do cliente.

Sobre a Empresa

A Fábrica de Códigos, fundada em 2015 em Marília, São Paulo, é uma empresa especializada em soluções de tecnologia da informação, focada no desenvolvimento de sistemas para facilitar a gestão empresarial. Seus principais produtos incluem o Shopping de Preços, uma plataforma que integra mais de 100 marketplaces e sistemas de e-commerce, e o Sysemp ERP, um software de gestão empresarial voltado para diversos setores como varejo e distribuição. Além disso, a empresa oferece um serviço gratuito de DNS dinâmico (DDNS) para acesso remoto a dispositivos. Com um ambiente de trabalho colaborativo e foco na inovação, a Fábrica de Códigos também é reconhecida por seu compromisso com o desenvolvimento profissional e sua participação em eventos educativos, como o Fábrica de Projetos na Universidade de Marília.

Sobre o Projeto

Este projeto consiste em um sistema integrado para consulta e gerenciamento de produtos via código de barras, desenvolvido com uma API em C# e banco de dados SQL, combinado a uma interface web responsiva. A solução permite uma leitura rápida de códigos, consulta detalhada de produtos e gestão segura de usuários, com diferentes níveis de acesso. Desenvolvido com metodologia Kanban, o sistema prioriza usabilidade, desempenho e escalabilidade, sendo ideal para otimizar processos em ambientes comerciais e de estoque. A arquitetura técnica combina o robusto backend em C# com um frontend dinâmico em JavaScript, potencialmente utilizando Vue.js para maior interatividade.
